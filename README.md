# 🚀 Two-Tier Application (Docker + MySQL)

## 📌 Project Overview
This is a **Two-Tier Application** built using Docker and Docker Compose.  
It demonstrates containerized deployment of a backend application connected with a MySQL database.

---

## 🏗️ Architecture

- **Frontend/Backend App Container**
- **MySQL Database Container**
- Communication handled via Docker Network

---

## ⚙️ Tech Stack

- Docker 🐳
- Docker Compose
- Node.js / Express (or your backend tech)
- MySQL Database
- WSL (Ubuntu)
